# （36）- 作用域

[toc]

## 一、作用域

### 1、作用域的定义

  作用域，就是变量的有效范围。除了作用域，变量就不可使用了。作用域包含但不限于一下几个点：

  1）所有文件都可以使用的变量；

  2）当前文件可以使用的变量；

  3）函数内部才能使用的变量；

  4）语句块内部才可以使用的变量；

### 2、作用域的决定因素

  决定变量作用域的因素，是变量定义的位置。

## 二、作用域的应用

### 1、全局变量的作用域

  对于全局变量（或者全局函数），它的作用域从声明它的地方开始，直到文件末尾结束。如下代码所示：

```c
#include <stdio.h>

int main() {
    return 0;
} 

int x;
void gogogo() {
	++x;
}
```

  在`gogogo()`函数中，不需要声明，就可以使用全局变量 x，但是这个变量不能用在 `main()`函数中，同样`gogogo()`这个函数也不能被`main()`调用。
